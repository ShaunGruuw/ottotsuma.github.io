Calculations

HP = (level*10)+(Vit*14.5) = 10000
MP = (level*5)+(Mag*8) = 6840

This is not an ideal choice for 1v1 combat, as the str is low, the hp is really low.
Let's have it fight A slow two handed warrior, base stats are:
Warrior, Str 600, Vit 600, def 100, dex 0
BladeSinger, 300, 0, 0, 803

With equipment, assuming the blade slinger moves 8x the speed, they use the same skill (50 power)
the bladeslinger gets 1 skill hit, 7 normal hits, 1 skill hit
warrior in the same time gets 2 skills hits
warrior has 98.5% hp left, bladedancer 91.2%
this doesn't even account for weapon power, different skills, different equipment.
all this is showing is that 1v1, all things equal, the warrior will win.

Even with the rouge type using the 50 power skill more often, and both using 80 power on the 8th.
The gap just gets larger as time goes on.
eg, 50, 1 ,1, 50, 1 ,1 ,1, 80 = 481, the warrior doing 50, 80, = 1150.
